# CHANGELOG - campus-hub 变更日志

## [Unreleased]

### 2026-01-20

- 新增搜索功能：支持搜索帖子（标题+正文）和用户（昵称）。
- 新增站内通知系统：
  - 被评论通知：帖子收到新评论时通知作者
  - 被回复通知：评论被回复时通知评论者
  - 新粉丝通知：被关注时收到通知
  - 点赞通知：帖子/评论被点赞时收到通知
- Header 新增通知铃铛图标，显示未读数量。
- 新增通知列表页面（/notifications）。

### 2026-01-19

- 新增用户头像/封面/简介字段与编辑接口。
- 新增关注/粉丝统计与列表接口。
- 新增用户评论列表接口。
- 举报管理后台（/admin）与举报弹窗。
- 个人主页支持关注弹窗与评论列表。

- 社区论坛基础功能（板块、帖子、评论）。
