## 项目需求（更新至 2026-01）

### 已完成

- 账号体系（注册/登录）
- 帖子/评论基础功能（列表、详情、发布、删除）
- 富文本编辑器 + 图片上传
- 举报入口 + 管理员处理
- 用户主页（头像/封面/简介）
- 关注/粉丝统计 + 列表
- 聊天室（WebSocket）
- 搜索（帖子标题/正文、用户昵称）
- 站内通知（被评论/被回复/被关注/被点赞）
